I mainly work with JavaScript, [TypeScript](https://www.typescriptlang.org/), [Rust](https://www.rust-lang.org/), C++, [SvelteKit](https://kit.svelte.dev/), [Tauri](https://tauri.app/), and [Godot](https://godotengine.org/).
